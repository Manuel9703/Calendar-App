# Calendar-App

A Vite + React + Tailwind PWA for tracking work shifts and estimating gross pay.

## Scripts

- npm install
- npm run dev
- npm run build

## Deployment

The app is configured for GitHub Pages via the workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml). Update the base path in [vite.config.js](vite.config.js) from /NOME-REPO/ to your repository name before the first deploy.
